Automated Sales Data Cleaning & Analysis
🚀 Overview

This project presents an end-to-end automated data cleaning pipeline built using Python. It processes raw sales data, handles inconsistencies, and generates meaningful visual insights through exploratory data analysis (EDA).

The goal is to transform messy, real-world data into a clean and analysis-ready dataset.

⚙️ Features


🔹 Data Cleaning Automation
Removes duplicate records
Standardizes column names
Handles missing values (mean, median, mode)
Detects & treats outliers using IQR method
Replaces invalid values (negative values, incorrect entries)
Cleans categorical data (trimming, formatting)



🔹 Data Transformation
Converts date columns into proper datetime format
Normalizes numerical values
Fixes inconsistent categorical labels



🔹 Logging System
Tracks all cleaning steps in Cleaning.log
